# Landing site (Astro)

Public page for **New Gen Agent**, styled after research sites like
[Nous Research](https://nousresearch.com/) — dark UI, terminal metric blocks,
typography-first layout.

## Local preview

```bash
cd site
npm install
npm run dev
```

Open the URL printed by Astro (usually `http://localhost:4321/new-gen-agent/`).

## Build

```bash
cd site
npm run build
npm run preview
```

Output goes to `site/dist/` and is published by GitHub Pages
(`.github/workflows/pages.yml`).
